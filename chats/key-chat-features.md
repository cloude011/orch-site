# Key Chat Features

To help you manage and organize your communications effectively, Orchestra offers a range of actions you can perform with chats. These features are designed to give you control over how you interact with different chats, ensuring that your conversations are streamlined and accessible.

Here are the **main actions you can perform with chats**:

1. Make a chat favorite;
2. Mark a chat as read or unread;
3. Add or remove members from a chat;
4. Mute a chat;
5. Pin and unpin chats in the chat list;
6. Archive and unarchive chats.

## Make Chat Favorite;

Any chat can be marked as a favorite. Once marked, the chat will appear in the Favorite folder in the[Chats Page](/pages-and-ui-elements/chats-page.md#chat-list). You can **mark a chat as favorite** in the following ways:

* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list):
  * Find the chat you want to mark as a favorite;
  * Right-click on the chat;
  * In the opened context menu select `Add to favorite`;
* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-header):
  * Open the chat you want to mark as a favorite;
  * Right-click on the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-header);
  * In the opened context menu select `Add to favorite`;
* From the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md)
  * Left-click the `...` icon in the top right corner of the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md);
  * In the opened context menu select `Add to favorite`.

To **remove a chat from favorites**, follow the same steps, but choose `Remove from favorite` from the context menu.

## Mark Chat as Read

You can **mark any chat with unread messages as read** without having to actually open it. You can do this with a certain chat or with a group of chats. Here's how:

To **mark a single chat as read**:

* In the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list), find the chat with unread messages;
* Right-click on the chat;
* In the opened context menu select `Mark as read`.

Also, you can **mark a group of chats as read** using the [Chats Page](/pages-and-ui-elements/chats-page.md#unread-folder) in the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list):

* Open the Unread folder;
* Right-click on the block where you want to mark chats as read;
* In the opened menu select `Mark as read`.

## Mark Chat as Unread

Here is how you can **mark a chat as unread**:

* In the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list) find the chat you want to mark as unread;
* Right-click on the chat;
* In the opened context menu select `Mark as unread`.

An **unread indicator** will appear on chats marked as unread.

## Mute Chat

You can mute any chat in Orchestra. For muted chats, notifications about new messages are not displayed. Additionally, the unread message count will appear in gray.

You can **mute a chat** in two ways:

* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list):
  * Find the chat you want to mute;
  * Right-click on the chat;
  * In the opened context menu select `Mute`;
* From the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md):
  * Left-click the `...` icon in the top right corner of the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md);
  * In the opened context menu select `Mute`.

To **unmute a chat**, follow the same steps, but choose `Unmute` from the context menu.

## Pin Chat in Chat List

You can **pin a chat** to the top of your [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list) for easy access. There is no limit to the number of chats you can pin. Here's how:

* Find the chat you want to pin;
* Right-click on the chat;
* In the opened context menu select `Pin`.

To **unpin a chat**, follow the same steps, but select `Unpin` from the context menu.

## Archive Chat

You can archive any chat to remove it from your main chat list but still retain it for later access.

There are several ways to **archive a chat**:

* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list):
  * Find the chat you want to archive;
  * Right-click on the chat;
  * In the opened context menu select `Archive`;
* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-header):
  * Open the chat you want to archive;
  * Right-click on the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-header);
  * In the opened context menu select `Archive`;
* From the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md):
  * Left-click the `...` icon in the top right corner of the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md);
  * In the opened context menu select `Archive`.

## Unarchive Chat

There are several ways to **unarchive a chat** and restore it to the main chat list:

* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-list):
  * Find the archived chat;
  * Right-click on the chat;
  * In the opened context menu select `Unarchive`;
* From the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-header):
  * Open the archived chat;
  * Right-click on the [Chats Page](/pages-and-ui-elements/chats-page.md#chat-header);
  * In the opened context menu select `Unarchive`;
* From the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md):
  * Left-click the `...` icon in the top right corner of the [Info Sidebar](/pages-and-ui-elements/info-sidebar.md);
  * In the opened context menu select `Unarchive`.
